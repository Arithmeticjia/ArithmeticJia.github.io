Seq2seq模型及注意力机制模型

发表于 2019-12-22 阅读次数：
本文字数： 1.5k 阅读时长 ≈ 1 分钟

对弈处理输出序列为不定长情况的问题呢，例如机器翻译，例如英文到法语的句子翻译，输入和输出均为不定长。前人提出了seq2seq模型，basic idea是设计一个encoder与decoder，其中encoder将输入序列编码为一个包含输入序列所有信息的context vector $ c $，decoder通过对$ c $的解码获得输入序列的信息，从而得到输出序列。encoder及decoder都通常为RNN循环神经网络
